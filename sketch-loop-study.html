A quiet corner, weather falling.

Now each stroke is actually drawn: when a spot in the noise field crosses the ink threshold, the stroke traces itself from one end to the other. When the field drifts past, the stroke un-draws itself in reverse. So it always feels like an invisible hand scribbling ahead of you.

Tweak stroke width for pencil vs. marker. Tweak draw time for calm vs. frantic hand. Each stroke has a slightly different speed, so a dense clump resolves with a bit of shimmer instead of one synchronous flash.
